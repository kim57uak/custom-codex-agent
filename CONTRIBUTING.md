# Contributing to Custom Codex Agent Platform

먼저 이 프로젝트에 관심을 가져주셔서 감사합니다! 여러분의 기여는 이 프로젝트를 더욱 가치 있게 만듭니다.

## How Can I Contribute?

### 1. Reporting Bugs
*   [Issue Tracker](https://github.com/your-username/custom-codex-agent/issues)를 사용하여 버그를 제보해 주세요.
*   버그를 재현할 수 있는 단계, 예상 결과, 실제 결과를 명확히 기술해 주세요.
*   스크린샷이나 에러 로그를 첨부하면 큰 도움이 됩니다.

### 2. Suggesting Enhancements
*   새로운 아이디어나 기능 개선 제안은 Issue를 통해 공유해 주세요.
*   해당 기능이 왜 필요한지, 어떤 가치를 제공하는지 설명해 주세요.

### 3. Pull Requests
*   새로운 브랜치를 생성하여 작업해 주세요. (`feat/new-feature` or `fix/bug-id`)
*   [Karpathy Coding Guidelines](docs/GUIDELINES.md)를 준수해 주세요 (단순함 유지, 불필요한 추상화 배제).
*   PR을 제출하기 전에 로컬에서 테스트를 완료했는지 확인해 주세요.

## Coding Style
*   **Python**: PEP 8을 준수하며, `Black` 포맷터를 권장합니다.
*   **Javascript**: Vanilla JS의 명료함을 유지해 주세요.
*   **CSS**: 변수 기반의 테마 시스템을 존중하며 수정해 주세요.
